# ポケモンさんすうゲーム

ポケモンを捕まえながら算数（足し算など）を学べるブラウザゲームです。

---

## 🎮 ゲーム内容

- ランダムでポケモンが出現
- 足し算の問題に挑戦
- 正解するとモンスターボールで捕獲チャンス
- 捕まえたポケモンは図鑑に登録される
- 5回以内で必ず捕獲チャンスあり

---

## 📱 対応環境

- iPhone / iPad
- PCブラウザ
- Safari推奨
